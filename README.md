# 📊 Diff Checker

> 文本差异对比工具 — 并列视图 · 统一视图 · 单词级高亮

## ✨ 功能

| 模块 | 功能 |
|---|---|
| 📊 **并列视图** | 左右双栏对比，新增（绿色）/ 删除（红色）/ 修改（黄色）分行高亮 |
| 📋 **统一视图** | 类 Git diff 格式（`-` 删除 / `+` 新增 / `~` 修改），集中查看差异 |
| 🔤 **单词级高亮** | 修改行内精确标注增删的单词，而非整行标红 |
| 🔄 **交换对比** | 一键交换左右文本，方便反向对比 |
| 📈 **差异统计** | 实时统计新增、删除、修改、相同行数 |
| ⚙️ **选项** | 忽略大小写、忽略空白差异、单词级高亮开关、自动换行 |
| 📝 **示例数据** | 一键加载 JavaScript 代码对比示例 |

## 🛠 技术栈

- 纯 HTML5 + CSS3 + JavaScript
- LCS（最长公共子序列）差分算法
- 零外部依赖

## 🚀 部署

支持 Vercel / Netlify / Cloudflare Pages 等静态托管平台。

```bash
npx vercel --prod
```

## 🔒 隐私

所有处理在浏览器端完成，**文本不上传服务器**。
