# tools-diff-checker
### 包含功能

| 模块 | 能力 |
|---|---|
| 📊 **并列视图** | 左右两栏对比，新增/删除/修改行分别高亮 |
| 📋 **统一视图** | 类 Git diff 格式，差异行集中展示 |
| 🔤 **单词级高亮** | 修改行内精确标注增删的单词 |
| 🔄 **交换对比** | 一键交换左右文本 |
| 📈 **差异统计** | 实时统计新增/删除/修改/相同行数 |
| ⚙️ **选项** | 忽略大小写、忽略空白、单词级高亮开关、自动换行 |
| 📝 **示例数据** | 一键加载 JavaScript 代码对比示例 |

**底层算法：** LCS（最长公共子序列）行级差分 + 单词级逐词差分
